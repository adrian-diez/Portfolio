Class Royale

Aplicación de gestión de aulas con componente de gamificación

La idea de Class Royale surge como respuesta a una problemática existente en mi antiguo trabajo como profesor de instituto.

En las clases del instituto se notaba una fuerte desmotivación, provocada por la falta de objetivos. Algunos profesores habían intentado utilizar técnicas de motivación con los alumnos, sin conseguir grandes resultados.
A ello se suma la dificultad de uso de programas de gestión interna del instituto, que desanimaban a los profesores más veteranos a usarlos. Como solución, se propuso una aplicación de gestión de aulas de interfaz sencilla que actuase, por una parte, como lugar donde guardar las tareas, citas y demás; por otra parte, se muestra un sistema de puntos al alumnado para generar una motivación externa para alcanzar los conocimientos.

La aplicación, hecha íntegramente por un compañero de estudios y yo, utiliza Vue.js para organizar el front end en vistas y componentes; reforzado por la librería Nuxt.js que simplifica la conexión con el back end. A este propósito, se utilizó Mongoose y Express.js para construir un esquema de datos, dividido en modelo, ruta y controlador, de la base de datos. Se decidió que la base de datos sería no relacional para tener más libertad a la hora de elegir cómo se estructurarían los datos.

En cuanto al componente visual, se usó Figma para un primer diseño de las principales vistas, y Vuetify para llevarlas a cabo en el código.
